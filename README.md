Animal Gallery

This project is a simple, clean visual gallery that displays a collection of high-resolution animal photographs.
It is built using HTML and CSS and deployed using GitHub Pages.
The purpose of this project is to practice layout design, asset organization, and static site deployment.

Live Demo

https://khushiagarwal0808.github.io/animal-gallery/

Features

Full-width responsive image layout

Clean and minimal design

Organized project structure

Deployed using GitHub Pages

Easy to extend with more images or sections
